#Creates a tendril cube

Cube c

Procedure tendrilHelper
Instance c
Rotate X [357,363]
Rotate Y [357,363]
Rotate Z [357,363]
Translate 3 0 0

Procedure tendril
Do tendrilHelper [5,20]

Procedure rowHelper
Add tendril
Translate 0 3 0

Procedure row
Do rowHelper 10

Procedure cubeHelper
Add row
Translate 0 0 3

Procedure cube
Do cubeHelper 10

Evaluate cube output
